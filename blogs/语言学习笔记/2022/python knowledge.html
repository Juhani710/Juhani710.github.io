# 关于编码

  • ASCII编码只有127个字符，包括字母数字与一些符号，用一个字节表示一个字符。
  • Unicode字符集把所有语言整合到一套编码里，包括中文等。常用的USC-16编码用两个字节表示一个字符。
  • 新的问题是用Unicode编码比ASCII编码需要多一倍的存储空间，在存储和传输上就十分不划算。所以，本着节约的精神，又出现了把Unicode编码转化为“可变长编码”的UTF-8编码。UTF-8编码把一个Unicode字符根据不同的数字大小编码成1-6个字节，常用的英文字母被编码成1个字节，汉字通常是3个字节，只有很生僻的字符才会被编码成4-6个字节。如果你要传输的文本包含大量英文字符，用UTF-8编码就能节省空间。
  • 在计算机内存中，统一使用Unicode编码，当需要保存到硬盘或者需要传输的时候，就转换为UTF-8编码。用记事本编辑的时候，从文件读取的UTF-8字符被转换为Unicode字符到内存里，编辑完成后，保存的时候再把Unicode转换为UTF-8保存到文件。浏览网页的时候，服务器会把动态生成的Unicode内容转换为UTF-8再传输到浏览器。

# 字典dict

  • key-value存储，查找速度极快，但空间占用多。表示：d = {'Michael': 95, 'Bob': 75, 'Tracy': 85}，查找：d['Michael']。注意dict内部存放的顺序和key放入的顺序是没有关系的。
  • 把数据放入dict的方法，除了初始化时指定外，还可以通过key放入：d['Adam'] = 67
  • 通过in可以判断key是否存在于dict中，如 'Thomas' in d 。或者通过dict提供的get()方法，没有则返回None或自己指定的值。
  • 删除一个key（及其对应value），用pop(key)方法
  • dict的key不可变。字符串、整数等都是不可变的，因此，可以放心地作为key。而list是可变的，就不能作为key。

# set

  • set和dict类似，也是一组key的集合，但不存储value。由于key不能重复，所以，在set中，没有重复的key。要创建一个set，需要提供一个list作为输入集合：s = set([1, 2, 3]) => s= {1,2,3}
  • add(key)方法可以添加元素到set中
  • remove(key)方法可以删除元素
  • set可以看成数学意义上的无序和无重复元素的集合，因此，两个set可以做数学意义上的交集、并集等操作。如 s1 & s2 , s1 | s2。

# 函数的参数

  1. 位置参数：根据位置一一对应
  2. 默认参数：可在定义函数时设置一个默认值，调用时可以不传入参数。注意默认参数必须指向不变对象，否则像列表这种对象每次调用都会改变默认值
  3. 可变参数：定义时使用 *+列表或元组名，表示传入其内的所有元素。调用时也这样用。
  4. 关键字参数：定义时使用 **kw，传入参数为关键字参数，格式为“关键字=值”，在函数内部组装成dict（名字是kw）。也可以在调用时直接传入一个已存在的dict的内容，格式为“**dict名”。
  5. 命名关键字参数：用以限制关键字参数的名字，命名关键字参数需要一个特殊分隔符*，后面的参数被视为命名关键字参数。而如果函数定义中已经有了一个可变参数，后面跟着的命名关键字参数就不再需要一个特殊分隔符了。命名关键字参数必须传入参数名，这和位置参数不同。格式如下所示：def person(name,age,*,city='Beijing",job) 调用时 person('Jack',24,job='Engineer')
  6. 各种参数的类型组合起来，位置参数作为必选参数必须传入，可变参数和关键字参数可以传入的数量为0。对于任意函数，都可以通过类似func(*args, **kw)的形式调用它，无论它的参数是如何定义的。

# 列表生成

  • 切片：取list或tuple的部分元素（切片操作符）L[0:6:2],表示从第一个元素开始，到第6个元素（不包含下标为6的元素），每两个取一个
  • 列表生成式：[ 要生成的表达式 for 变量 in 可迭代对象 if 筛选条件 ] 如 [x * x for x in range(1, 11) if x % 2 == 0]。还可以用两层循环，以及用两个变量来生成list。如
d = {'x': 'A', 'y': 'B', 'z': 'C' }
[k + '=' + v for k, v in d.items()]
# ['y=B', 'x=A', 'z=C']

列表生成式for循环的后面只能跟上if而不能带else，其表示筛选条件。而for的前面可以写if，此时必须加else，此时表示根据变量的特征做不同的处理，最后必须是一个确定的表达式。

  • 生成器generator：一边循环一遍计算list的元素，不必创建完整list用以节省空间。将列表生成式的[]改成()，即创建了一个generator对象。

可以用next(generator对象)这一函数不断获得生成器的下一个返回值。直到计算到最后一个元素，没有更多的元素时，抛出StopIteration的错误。另外由于generator也是可迭代对象，可以用for循环不断打印下一个值。 定义generator的另一个方法时创建一个generator函数，使用yield关键字。

def fib(max):
	n, a, b = 0, 0, 1
	while n < max:
		yield b
		a, b = b, a + b
		n = n + 1
	return 'done'

遇到yield语句就返回值，再次执行时从上次返回的地方继续。

  • 可迭代对象（Iterable）：可以直接作用于for循环，包括list，tuple，dict，set，str等集合数据类型与generator。

可以用isinstance()判断一个对象是否是Iterable对象，需从collections.abc引入模块Iterable

  • 迭代器（Iterator）：可以被next()函数调用并不断返回下一个值的对象。也可以用上述方法判断。

把list、dict、str等Iterable变成Iterator可以使用iter()函数。

# 函数式编程

  • 几种函数
  1. map()函数接受两个参数，一个是函数，一个是Iterable，map将传入的函数依次作用到序列的每个元素，并把结果作为新的Iterator返回。注意返回值是惰性序列Iterator，可以通过list()函数让其把整个序列计算出来并返回一个list
  2. reduce()接受两个参数，也是一个函数和一个序列，而接受的这个函数必须接受两个参数，也就是序列的前两个参数，并把结果与序列的下一个元素做累计计算。效果为reduce(f, [x1, x2, x3, x4]) = f(f(f(x1, x2), x3), x4)
  3. filter()函数用于过滤序列。与map()类似，接收一个函数与一个序列，不同的是接收的函数需要是一个判断的函数，把传入的函数依次作用于每个元素，然后根据返回值是true还是False决定保留还是丢弃该元素。
  4. sorted()排序函数，接收一个list进行排序（升序），还可以接收一个key函数来实现自定义排序，例如在参数中令key=abs即可按绝对值大小排序。（key指定的函数将作用于list的每一个元素上，并根据key函数返回的结果进行排序）另外要进行反向排序，不必改动key函数，可以传入第三个参数reverse=True。
  • 匿名函数lambda：简小精悍的函数

lambda 参数：返回值

  • Decorator装饰器：在代码运行期间动态增加功能，本质上是一个能返回函数的高阶函数

函数对象可以被赋值给变量，通过变量也能调用函数。函数对象有一个__name__属性，可以拿到函数的名字。

import functools

def log(text):
    def decorator(func):
        @functools.wraps(func)
        def wrapper(*args, **kw):
            print('%s %s():' % (text, func.__name__))
            return func(*args, **kw)
        return wrapper
    return decorator

#使用
@log('execute')
def now():
    print('2015-3-25')
#效果等价于now = log('execute')(now)

这是一个完整的带参数的decorator。functools模块里的wraps()用于保持原始函数now的__name__属性不变（否则会变成wrapper）。首先执行log('execute')，返回的是decorator函数，再调用返回的函数，参数是now函数，返回值最终是wrapper函数。wrapper()函数的参数定义是(*args, **kw)，因此，wrapper()函数可以接受任意参数的调用。在wrapper()函数内，首先打印日志，再紧接着调用原始函数。

# 面向对象编程

  • 创建类

class 类名 （继承的父类）： def init(self, 创建实例时需要的属性作为参数):
self.属性 = 参数
在属性前面加上__表示私有，外部不能再更改（实际上不能直接访问__name是因为Python解释器对外把__name变量改成了_Student__name，所以，仍然可以通过_Student__name来访问__name变量）

  • 继承可以获得父类的所有功能，但也可以重新定义父类的方法，获得多态的优点

  • 对方法的使用：

有三种方法 1.实例方法：由创建的实例进行调用，可以访问实例和类的属性，但修改属性只对当前实例有效。 2.类方法：由类或者实例进行调用，访问类的属性（对所有实例都有效），通过装饰器@classmethod进行修饰，函数第一个参数必须是关键词 clf 。 3.静态方法：静态方法第一行要有装饰器 @staticmethod ，函数参数绝对不能有关键词 clf 和 self，不涉及类或实例的属性。

静态语言vs动态语言 对于静态语言（例如Java）来说，如果需要传入Animal类型，则传入的对象必须是Animal类型或者它的子类，否则，将无法调用run()方 法。 对于Python这样的动态语言来说，则不一定需要传入Animal类型。我们只需要保证传入的对象有一个run(）方法就可以了

  • 动态绑定：可以给实例绑定属性，也可以给类绑定属性。还可以动态给实例或类绑定方法。
  • 在定义类时，可以定义一个特殊的__slots__变量，来限制该class实例能添加的属性
__slots__ = ('name', 'age') # 用tuple定义允许绑定的属性名称

slots_ 可以理解为给某个类预设属性清单，所有根据此类创建的实例只允许绑定清单内的属性，不能绑定其他属性。 但该清单只对本类有效，对其子类无效（除非其子类也有__slots__）。 若其子类也有__slots__，则子类实例可绑定的属性为父类清单和子类清单的并集。

  • @property装饰器

在类中利用@property装饰器创建只读属性，其会将方法转换为相同名称的只读属性。可以像调用属性一样调用这个方法，这个方法只能有self一个参数。在类中还可以以封装的形式对这个方法（属性）进行操作。

class Student(object):

    @property
    def score(self):
        return self._score #要注意这里return的属性名字不能与方法名相同，否则会无限递归

    @score.setter
    def score(self, value):
        if not isinstance(value, int):
            raise ValueError('score must be an integer!')
        if value < 0 or value > 100:
            raise ValueError('score must between 0 ~ 100!')
        self._score = value

这样就可以在外对属性进行赋值（同时可以进行检查）与读取。如果没有第二个方法，只有第一个就是一个只读的属性。

  • 通过修改特殊用途的方法定制类

1.类似__xxx__的属性和方法在Python中都是有特殊用途的，比如__len__方法返回长度。在Python中，如果你调用len()函数试图获取一个对象的长度，实际上，在len()函数内部，它自动去调用该对象的__len__()方法 2.如果用print打印一个实例，会调用__str__()方法。 3.如果一个类想被用于for ... in循环，类似list或tuple那样，就必须实现一个__iter__()方法，该方法返回一个迭代对象，然后，Python的for循环就会不断调用该迭代对象的__next__()方法拿到循环的下一个值，直到遇到StopIteration错误时退出循环。以斐波那契数列为例：

class Fib(object):
    def __init__(self):
        self.a, self.b = 0, 1 # 初始化两个计数器a，b

    def __iter__(self):
        return self # 实例本身就是迭代对象，故返回自己

    def __next__(self):
        self.a, self.b = self.b, self.a + self.b # 计算下一个值
        if self.a > 100000: # 退出循环的条件
            raise StopIteration()
        return self.a # 返回下一个值

4.getitem()方法可以像list那样按下标取元素

class Fib(object):
    def __getitem__(self, n):
        a, b = 1, 1
        for x in range(n):
            a, b = b, a + b
        return a

如果加上切片的功能需要这个方法判断传入的参数是一个int还是一个切片对象slice

class Fib(object):
    def __getitem__(self, n):
        if isinstance(n, int): # n是索引
            a, b = 1, 1
            for x in range(n):
                a, b = b, a + b
            return a
        if isinstance(n, slice): # n是切片
            start = n.start
            stop = n.stop
            if start is None:
                start = 0
            a, b = 1, 1
            L = []
            for x in range(stop):
                if x >= start:
                    L.append(a)
                a, b = b, a + b
            return L

5.当调用不存在的属性时，getattr()方法可以动态返回一个属性。当调用不存在的属性时，比如score，Python解释器会试图调用__getattr__(self, 'score')来尝试获得属性，这样，我们就有机会返回score的值：

class Student(object):

    def __init__(self):
        self.name = 'Michael'

    def __getattr__(self, attr):
        if attr=='score':
            return 99

6.call()方法的定义可以直接对实例进行调用，此时这个实例就是一个函数

class Student(object):
    def __init__(self, name):
        self.name = name

    def __call__(self):
        print('My name is %s.' % self.name)

可以直接调用 instance()，call()方法的定义也可以设置参数。
