# 搭建

网站完全采用vuepress-theme-reco提供的vuepress主题框架，网站服务运行在github提供的Github-Page服务中，域名从阿里旗下的万网上购得

# 首页元素

首页包含最上方以及中部的展示页面用以欢迎您的到来，来到下部即可看到本人更新过的历史博客的记录。导航栏包括一个搜索框，categories可以分类别访问个人的博客，tags则可以通过标签分类查看。技术支持下拉框包括本网站搭建参考的vuepress-theme-reco主题的介绍网站以及网站托管在github上的仓库页面。
